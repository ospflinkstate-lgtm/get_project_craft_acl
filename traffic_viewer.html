📘 Network Classification Logic (เกณฑ์การแยกประเภท)

🏠 Internal Network

Logic: IP ที่ขึ้นต้นด้วย 10.x, 172.16.x หรือ 192.168.x
Note: เป็นทราฟฟิกจากอุปกรณ์ภายในองค์กร หรือเครื่องใน Subnet อื่นที่เชื่อมต่อเข้ามาทาง Internal Link

☁️ CDN / Cloud Nodes

Logic: ตรวจสอบจาก IP Prefix ของผู้ให้บริการ (เช่น Cloudflare, Google, AWS) และการทำ Reverse DNS หา Hostname ที่มีคำว่า cache หรือ mirror
Note: ทราฟฟิกกลุ่มนี้มักเป็นบริการอัปเดตไฟล์ หรือบริการเว็บยอดนิยม

🌍 External (Other)

Logic: IP สาธารณะทั่วไปที่ไม่ตรงกับฐานข้อมูล CDN หรือ ISP Node ที่ระบุไว้
Note: ควรตรวจสอบ IP กลุ่มนี้เป็นพิเศษ หากมีค่า Intensity สูงแต่ไม่มีชื่อ Domain ระบุไว้
